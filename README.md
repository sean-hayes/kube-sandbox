# A sandbox of a CoreOS cluster with a Kubernetes control plane running in Vagrant

#### Architecture

 * CoreOS cluster
 * Kubernetes control plane
  * Hello World node.js service
  * DataZoomer service
   * Apache pod
   * Mariadb pod
   * Persistant storage
   * configuration via etcd
   * [https://prometheus.io/]
